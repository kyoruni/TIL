# リダイレクト

標準入出力、標準エラー出力の入力先を変更することを、リダイレクトという。

## 入力先の指定


```bash
# 入力元になるファイル
$ cat hoge.txt

hoge
```

```bash
$ cat < hoge.txt

hoge
```

## 出力先の指定

```bash
# ls -la の内容を、hoge.txtに書き込む
$ ls -la > hoge.txt
```

- `>` 上書き保存

- `>>` 追加保存

## 標準エラー出力のリダイレクト

```bash
# ないディレクトリの情報を hoge.txt に書き込もうとする
$ ls /hoge > hoge.txt
ls: /hoge: No such file or directory

# hoge.txt の内容を確認 → 標準出力には何もないため、中身がない
$ cat hoge.txt

```

- `2>` 標準エラー出力のリダイレクト

```bash
# ないディレクトリの情報を hoge.txt に書き込もうとする
$ ls /hoge 2> hoge.txt 

# hoge.txt の内容を確認 → 標準エラー出力の内容が書き込まれている
$ cat hoge.txt

ls: /hoge: No such file or directory
```
