# コマンドラインの操作

## カーソル

### 1つ前へ

- `Ctrl + b`

  - `b` は 「Backword」 の B 

### 1つ後へ

- `Ctrl + f`

  - `f` は 「Forward」 の F

### 行頭へ

- `Ctrl + a`

### 行末へ

- `Ctrl + e`

### 単語単位で1つ前へ

- `option + ←`

### 単語単位で1つ後へ

- `option + →`

## 文字の削除

### 1文字削除

- `Ctrl + h`

### 1単語削除

- `Ctrl + w`

## カット & ペースト

### カーソル位置から行末までを削除（カット）

- `Ctrl + k`

### カーソル位置から行頭までを削除（カット）

- `Ctrl + u`

### 直前にカットした内容をペースト

- `Ctrl + u`
