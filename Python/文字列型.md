# 文字列型

## フォーマット文字列

- 文字列の前に `f`

  - `{}` で変数を埋め込む

##### fなし

```python
name = 'ほげ'
print('こんにちは {name} さん')

こんにちは {name} さん
```

##### fあり

```python
name = 'ほげ'
print(f'こんにちは {name} さん')

こんにちは ほげ さん
```

## row 文字列

- 文字列の前に `r`

- 文字列内のエスケープ文字をそのまま解釈する

##### 普通の文字列型

- `\\` を出力させたい場合、 4個書く必要がある

```python
hoge = 'C:\\Windows\\hoge'
print(hoge)

C:\Windows\hoge
```

##### row 文字列

- 書いてある通りに解釈する

```python
hoge = r'C:\\Windows\\hoge'
print(hoge)

C:\\Windows\\hoge
```
