# リスト

## リストから新しいリストを生成する

```python
numbers = [1, 2, 3, 4, 5]

# リストの中身全てに1を足す
data = [number + 1 for number in numbers]
print('data >>>>', data)
```

```python
data >>>> [2, 3, 4, 5, 6]
```

↑ と同じ

```python
numbers = [1, 2, 3, 4, 5]

# リストの中身に1を足す
data   = []
for number in numbers:
    data.append(number + 1)
print('data >>>>', data)
```

```python
data >>>> [2, 3, 4, 5, 6]
```
