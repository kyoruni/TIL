# フォルダ構成

- app
  - コントローラ、モデルなど

- bootstrap
  - フレームワークの起動やオートローディングの設定など

- config
  - 設定ファイル

- database
  - データベースのマイグレーションファイル
  - Seedファイル

- public
  - フロントコントローラとアセット（画像、CSSなど）

- resources
  - ビューやアセットの元ファイル
  - 言語ファイル

- routes
  - ルート定義

- storage
  - `/app` アプリケーションによって使用されるファイル
  - `/framework` フレームワークが生成するファイル
  - `/logs` ログファイル

- tests
  - 自動テスト

- vendor
  - Composerでインストールしたパッケージ
