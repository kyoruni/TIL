# concat 文字列連結

```
concat(1つ目のカラム, 2つ目のカラム)
```

##### 使ってみる

```
mysql> select * from hoge;
+----------+-----------+
| str1     | str2      |
+----------+-----------+
| 織田      | 信長      |
+----------+-----------+
1 rows in set (0.00 sec)


mysql> select str1, str2, concat(str1, str2)
    -> from hoge;
+----------+-----------+-----------------------+
| str1     | str2      | concat(str1, str2)    |
+----------+-----------+-----------------------+
| 織田      | 信長      | 織田信長                |
+----------+-----------+-----------------------+
1 rows in set (0.00 sec)
```
