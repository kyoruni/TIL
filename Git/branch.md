# branch

## ブランチ名変更

## 今いるブランチの名前を変更

##### hoge → fuga

```
$ git branch -m fuga
```

## 今いるブランチ以外の名前を変更

##### hoge → fuga

```
$ git branch -m hoge fuga
```

## ブランチを削除

```
$ git branch -d {branch name}
```

##### hogeブランチを削除

```
$ git branch -d hoge
```
