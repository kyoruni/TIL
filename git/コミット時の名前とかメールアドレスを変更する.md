# コミット時の名前とかメールアドレスを変更する

## 今何が設定されているか確認する

```
$ git config --local user.name
hoge
```

```
$ git config --local user.email
hoge@hoge.com
```

## 変更する

- local：今いるリポジトリの設定だけ変え

- global：今ログインしているユーザーの設定を変える

- system：こわいので使わない

```
$ git config --local user.name piyo

```

変わったか確認

```
$ git config --local user.name
piyo
```

```
$ git config --local user.email piyo@piyo.com
```

変わったか確認

```
$ git config --local user.email
piyo@piyo.com
```
